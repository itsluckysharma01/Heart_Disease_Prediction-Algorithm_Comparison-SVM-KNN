About Our AI System

Our heart disease prediction system combines the power of two proven machine learning algorithms to provide accurate and reliable health assessments.

Project Overview

This project implements and compares two powerful machine learning algorithms for heart disease prediction using the UCI Heart Disease dataset. Our system provides real-time predictions through an intuitive web interface.

UCI Heart Disease Dataset

Trained on the comprehensive UCI Heart Disease dataset containing 13 key medical features from over 1,000 patient records with verified diagnoses.

High Accuracy

Both algorithms achieve over 95% accuracy through careful feature engineering, data preprocessing, and hyperparameter optimization.

Machine Learning Algorithms

Support Vector Machine (SVM)

How it works:
  • Creates optimal decision boundaries between healthy and at-risk patients
  • Uses kernel functions to handle complex, non-linear relationships
  • Excellent at handling high-dimensional medical data
  • Robust against overfitting with proper regularization
Strengths:
  • High accuracy with complex datasets
  • Memory efficient
  • Effective in high-dimensional spaces
  • Versatile with different kernel functions

K-Nearest Neighbors (KNN)

How it works:
  • Compares new patients to the K most similar cases in training data
  • Makes predictions based on majority vote of nearest neighbors
  • No assumptions about data distribution
  • Adapts locally to data patterns
Strengths:
  • Simple and intuitive approach
  • No training period required
  • Works well with small datasets
  • Can handle multi-class problems naturally

Medical Features Analyzed

Patient Demographics

  • Age: Patient's age in years
  • Sex: Gender (Male/Female)

Cardiovascular Symptoms

  • Chest Pain Type: Typical angina, atypical angina, non-anginal, asymptomatic
  • Exercise Induced Angina: Whether exercise causes chest pain

Laboratory Tests

  • Cholesterol: Serum cholesterol level (mg/dl)
  • Fasting Blood Sugar: Whether > 120 mg/dl

Vital Signs

  • Resting Blood Pressure: In mm Hg
  • Maximum Heart Rate: Achieved during exercise

ECG Results

  • Resting ECG: Normal, ST-T abnormality, LV hypertrophy
  • ST Depression: Exercise-induced ST depression
  • ST Slope: Upsloping, flat, downsloping

Advanced Tests

  • Major Vessels: Number colored by fluoroscopy (0-3)
  • Thalassemia: Normal, fixed defect, reversible defect
